# Modular Configurations & Payload Integration

## 1. Engine Modularity (Thrust Scaling)
The AV-Apex is designed for modular manufacturing. The Apex Line and Fin geometry remain identical, but the thrust zones can be populated based on mission requirements:
* **Config A (2 Engines):** Uses Engines 1 & 2 (inner). Lowest cost, lowest speed, longest endurance. Used for localized agricultural surveying or border patrol.
* **Config B (4 Engines):** Uses Engines 1, 2, 3, 4. Mid-tier. Balanced speed and range for standard wildfire monitoring.
* **Config C (6 Engines):** Uses Engines 1–6. Maximum thrust stacking. Enables Mach 0.7+ cruise and the fastest 180° inversion arc. Used for long-range emergency SAR where time-to-target is critical.

## 2. External Payload Integration
Because the internal volume is 100% dedicated to fuel (F1–F7), all payloads must be external. The blended delta-wing underside features hardpoints that do not disrupt the Apex Line aerodynamics.
* **SAR Pod:** High-resolution optical/thermal cameras.
* **Atmospheric Sampler:** Air intake valves for volcanic or hurricane data collection.
* **Life-Raft Dispenser:** External deployable rafts for maritime rescue.
* **Communication Relay:** Extending Wi-Fi/SATCOM to disaster zones.
